# Інструкція зі встановлення APK

У цьому розділі описано процес завантаження та встановлення APK-файлу застосунку на Android-пристрій.

## 1. Завантаження APK

1. Перейдіть у розділ **Releases** цього репозиторію.
2. Відкрийте останній реліз.
3. У секції **Assets** знайдіть та завантажте файл з розширенням `.apk`.

Або скористайтеся прямим посиланням:

**➡️ Завантаження останньої версії:**
`https://github.com/IO-Codee/FitTrack/releases/download/v1.0.0/FitTrack_v1.0_release.apk`

---

## 2. Встановлення APK на Android

1. Після завершення завантаження відкрийте APK-файл на телефоні.
2. Якщо Android блокує встановлення:

   * відкрийте **Налаштування → Безпека** або **Конфіденційність**;
   * дозвольте встановлення застосунків із невідомих джерел для браузера або файлового менеджера.
3. Поверніться до APK-файлу та натисніть **Встановити**.
4. Дочекайтеся завершення встановлення та відкрийте застосунок.

---

## ⚠️ Важливо

Оскільки застосунок встановлюється не через Google Play, Android може показувати системне попередження безпеки. Це стандартна поведінка для APK, що поширюються через GitHub Releases.
